AUDIO

Audio este un tag ce se formeaza < audio src = > si are urmatoarele atribute prezentate mai pentru a putea controla elementul cum vrei noi

Si poate avea urmatorele formate ,WAV,MP3,OGG

Video

Video se formeaza la fel ca si audio ,doar ca are anumite atribute in plus precum:

  1. poster
  2. width & height

Source

Este un subelement ce poate avea doua formate diferite si poate alege pe cel potrivit pentru browser.

Are urmatoarele atribute

  • SRC
  • TYPE

TRACK

Este un subelement folosit pentru subtritrare in video

Si are urmatorele atribute:

  • SRC
  • KIND
  • SRCLANG(LIMBA)
  • LABEL

Iframe

Este un element pentru youtube upload video
